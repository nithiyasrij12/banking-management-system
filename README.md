# 🏦 Banking Management System (JDBC)

A console-based banking application developed using Java and JDBC that performs core banking operations like account creation, deposit, withdrawal, and balance check.

---

## 🚀 Features
- Admin and Customer login
- Create and delete customer accounts
- Deposit and withdraw money
- Check account balance
- Input validation and exception handling

---

## 🛠️ Tech Stack
- Java
- JDBC
- MySQL

---

## ⚙️ How to Run
1. Clone the repository
2. Configure database in DBConnection.java
3. Run BankingManagement.java

---

## 📌 Concepts Used
- JDBC (PreparedStatement)
- OOP (Encapsulation, Methods)
- Exception Handling
- Menu-driven program
